Token Pair and Arbitrage Screener

A Flask application to dynamically display cryptocurrency token pairs and identify Arbitrage opportunities using the Dexscreener API.

Key Features:

  • Dynamic HTML Generation: Generates HTML content dynamically to show token pairs and related details.
  • API Integration: Integrates with Dexscreener to fetch real-time token pair data.
  • Data Visualization: Uses matplotlib.pyplot to visualize token price trends and other relevant data.

Tools and Libraries Used:

  • Python 3.x
  • Flask for web development
  • requests for API calls
  • matplotlib.pyplot for charts
